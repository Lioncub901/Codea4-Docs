key.modifiers()¶

Queries the current key modifiers as a bit field, which is composed of the following bit mask constants:

  • key.leftAlt

  • key.rightAlt

  • key.alt

  • key.leftCtrl

  • key.rightCtrl

  • key.ctrl

  • key.leftCmd

  • key.rightCmd

  • key.cmd

Parameters

keyCode (constant) – The keyCode to query

Returns

Was the key released this frame

Return type

boolean
